The Industry AI Model Platform is an open-source, trusted LLM asset management platform that helps users manage the model library, datasets, user spaces, and development environments in a unified way, while providing LLMOps capabilities such as model inference, model training & evaluation, and resource management.

The main navigation areas of the platform are:

- `Model Library`: browse and manage model assets.
- `Model Inference`: includes the `Serverless APIs` and `Dedicated Instances` submenus.
- `Model Training & Evaluation`: includes the `Fine-tuning Instances` and `Model Evaluation` submenus.
- `Datasets`: browse and manage dataset assets.
- `Development Environment`: create and manage JupyterLab, VS Code, Eclipse Theia, and other development instances.
- `User Spaces`: manage application spaces and deployed apps.
- `Resource Management`: centrally view development environments, dedicated instances, fine-tuning instances, and model evaluation resources.
