## 创建入口

点击页面右上角的用户头像，在下拉菜单中选择 **新建数据集**，进入数据集创建页面。

## 填写数据集信息

在创建页面中填写以下信息，完成后点击 **创建数据集** 按钮：

| 参数 | 必填 | 说明 |
|------|------|------|
| **所有者** | 是 | 默认为当前登录用户；如果已加入组织，可从下拉列表中选择组织作为所有者 |
| **数据集英文名称** | 是 | 长度 2-64 个字符。必须以字母开头，以字母或数字结尾，仅可包含字母、数字、中划线（`-`）、下划线（`_`）和点号（`.`），且特殊字符不能连续出现 |
| **数据集别名** | 否 | 可选的友好显示名称，支持中文等字符 |
| **License** | 是 | 选择数据集的开源许可证类型 |
| **数据集简介** | 否 | 对数据集的简要文字描述 |
| **可见性** | 是 | 默认为 **私有**。公开数据集需要管理员审批后才可生效 |

{{< notice note >}}
数据集英文名称创建后不可修改，请谨慎填写。
{{</ notice >}}

## 完善数据集卡片

数据集创建成功后，页面会自动跳转到数据集详情页。建议按照 [数据集卡片](./dataset_card) 规范编辑仓库根目录下的 `README.md` 文件，补充数据集的详细描述、数据结构、使用方法等信息。

## 查看数据集列表

创建完成后，可通过以下方式查看数据集：

- **个人数据集**：点击用户头像，选择 **个人主页**，切换到 **数据集** 标签页查看个人创建的数据集
- **组织数据集**：进入对应组织页面，切换到 **数据集** 标签页查看组织下的数据集
- **全部数据集**：通过顶部导航栏的 **数据集** 入口浏览平台上所有公开的数据集
