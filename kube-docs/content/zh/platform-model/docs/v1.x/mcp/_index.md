## 什么是 MCP

Model Context Protocol（MCP）是一项开放协议，旨在标准化大型语言模型（LLMs）与外部数据源和工具的交互方式。它被比喻为"AI 应用的 USB-C 端口"——通过提供统一接口，简化 AI 模型与各种工具、数据库和服务的连接，解决传统 M×N 集成问题。

## 主要优势

- **门槛降低**：显著降低了大模型调用外部工具、软件和接口的复杂性。
- **生态系统扩展**：越来越多的软件和工具供应商加入 MCP 生态，丰富了可供 AI 模型使用的资源。
- **标准化对接**：提供统一接口，使大型语言模型能够轻松连接和使用不同的数据源和工具。

## 平台中的 MCP 仓库

平台提供 MCP 仓库（MCP Hub），用于托管和分享符合 MCP 标准的工具服务。开发者可以：

- 创建 MCP 服务仓库，发布自己开发的 MCP 工具。
- 浏览和获取社区开放的 MCP 工具服务。
- 将 MCP 工具部署为可运行的应用空间服务。

## MCP 应用部署

平台支持将 MCP 服务部署为独立的应用空间（Space），提供标准化的 API 接入点，供 AI 客户端（如支持 MCP 的智能体框架）直接调用。

## 典型应用场景

- **数据库查询**：通过 MCP 让大模型直接查询内部数据库。
- **文件读写**：让大模型读取或写入本地/远程文件系统。
- **API 调用**：将企业内部 API 封装为 MCP 工具，供大模型调用。
- **代码执行**：提供代码沙箱执行能力给大模型。
