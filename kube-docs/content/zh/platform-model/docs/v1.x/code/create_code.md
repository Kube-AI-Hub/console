## 入口

点击页面右上角头像，在下拉菜单中选择 **新建代码仓库**，进入创建表单页面。

## 填写表单

创建代码仓库时，需要填写以下参数：

| 参数 | 是否必填 | 说明 |
|------|----------|------|
| 所有者 | 是 | 默认为当前登录用户；如果已加入组织，可切换为组织名 |
| 代码仓库英文名 | 是 | 长度 2–64 个字符，以字母开头、字母或数字结尾，仅支持字母、数字、`-`、`_`、`.`，且不能连续使用 `-`、`_`、`.` |
| 别名 | 否 | 可选的中文名或友好名称 |
| 许可证 | 是 | 从下拉列表中选择合适的开源许可证 |
| 描述 | 否 | 简要描述代码仓库的用途 |
| 可见性 | 是 | **公开**：所有用户可见；**私有**：仅本人和组织成员可见 |

填写完成后，点击 **创建代码仓库** 按钮提交。

## 初始化仓库

创建成功后，平台会自动跳转至代码仓库详情页。建议首先编辑 `README.md` 文件，添加代码仓库的说明信息，例如项目简介、使用方法和依赖说明。

## 查看代码仓库列表

可通过以下方式查看已创建的代码仓库：

- **个人主页**：点击头像进入个人中心，切换到 **代码仓库** 标签页查看所有个人代码仓库。
- **组织主页**：进入对应组织页面，切换到 **代码仓库** 标签页查看组织下的代码仓库。
