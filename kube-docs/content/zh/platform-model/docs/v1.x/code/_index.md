## 什么是代码仓库

代码仓库（Code Hub）可存储模型和数据集相关联的代码，为开发者提供统一的代码托管平台。

## 核心功能

- **代码托管**：基于 Git 托管项目代码、训练脚本、推理代码等。
- **版本控制**：支持查看代码修改历史、对比差异、回滚到历史版本。
- **协作开发**：支持多人协作，通过分支和 Pull Request 管理代码变更。
- **权限控制**：支持公开和私有两种可见性设置。
- **在线浏览**：支持在 Web 界面直接浏览代码文件内容和目录结构。

## 适用场景

- 存储和版本化管理模型训练脚本
- 托管模型推理服务代码
- 存放与数据集配套的数据处理脚本
- 管理 AI 应用项目代码
