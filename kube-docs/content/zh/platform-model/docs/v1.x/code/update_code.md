## 编辑代码文件

平台支持以下两种方式编辑代码仓库中的文件：

### 本地 Git 编辑

通过 Git 克隆仓库到本地后，使用任意编辑器修改文件，再提交推送：

```bash
git clone https://<platform-host>/<namespace>/<code-repo-name>
cd <code-repo-name>

# 编辑文件后
git add .
git commit -m "Update code files"
git push origin main
```

### Web 在线编辑

1. 进入代码仓库详情页，点击 **文件** 标签页。
2. 点击需要编辑的文件名称，进入文件查看页面。
3. 点击编辑按钮，进入在线编辑模式。
4. 修改完成后，填写提交信息并保存。

## 修改仓库设置

进入代码仓库详情页，点击 **设置** 标签页，可以修改以下仓库信息：

| 设置项 | 是否可修改 | 说明 |
|--------|------------|------|
| 代码仓库路径 | 否 | 仓库创建后路径不可更改 |
| 别名 | 是 | 修改后点击 **更新别名** 按钮保存 |
| 描述 | 是 | 修改后点击 **更新描述** 按钮保存 |
| 可见性 | 是 | 可在 **公开** 和 **私有** 之间切换；切换时需要二次确认 |

## 删除代码仓库

1. 进入代码仓库详情页，点击 **设置** 标签页。
2. 滚动到页面底部的 **删除** 区域。
3. 输入代码仓库的完整路径（格式：`<namespace>/<repo-name>`）以确认删除。
4. 点击 **删除** 按钮。

{{< notice note >}}
删除操作不可逆，代码仓库及其所有文件、提交记录将被永久删除，请谨慎操作。
{{</ notice >}}
