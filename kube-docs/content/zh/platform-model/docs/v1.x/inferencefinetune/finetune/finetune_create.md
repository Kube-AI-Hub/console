## 创建入口

在想要微调的**基座模型**详情页，点击右上角**微调实例**按钮，跳转至创建页面。

{{< notice tip >}}
如果模型详情页未显示"微调实例"按钮，表示该模型当前不支持此功能。请联系平台管理员了解支持情况。
{{</ notice >}}

## 配置参数说明

进入微调实例创建页面后，填写以下配置信息，完成后点击**创建实例**：

| 参数 | 说明 |
|------|------|
| **实例名称** | 自定义名称，不能与已有实例重复，例如：`qwen-medical-finetune` |
| **模型 ID** | 平台中的模型标识，默认锁定当前选择的基座模型 |
| **区域/资源配置** | 根据基座模型规模选择合适的 GPU 资源配置（显存需能容纳模型和训练梯度） |
| **运行时框架** | 选择微调框架：**LLaMA-Factory** 或 **MS-Swift** |

## 查看实例列表

创建完成后，可在**资源控制台 → 微调实例列表**中实时查看实例启动进度，并管理已开启的微调任务。

## 使用微调框架

实例启动后，点击实例名称可进入微调框架界面：

- **LLaMA-Factory**：提供可视化的 LlamaBoard 界面，在页面上选择数据集、配置 LoRA 参数、启动训练，并实时查看 Loss 曲线等训练指标。
- **MS-Swift**：提供命令行和 Web 界面两种方式，支持更多模型类型和量化微调选项。

## 导出微调模型

训练完成后，可将微调权重合并并导出为新的独立模型仓库，用于后续的推理部署或评测。
