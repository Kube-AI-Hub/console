## 功能概览

资源管理模块为大模型全生命周期提供算力支撑。在最新导航中，相关能力分布在 `模型推理`、`模型训练评测`、`开发环境` 和 `资源管理` 菜单下，涵盖以下四大功能：

| 功能 | 说明 |
|------|------|
| [开发环境（Notebook 实例）](./notebook/notebook_intro) | 一键创建交互式开发环境，支持 JupyterLab、VS Code、Eclipse Theia |
| [模型推理](./inference/inference_intro) | 通过 `公共推理服务` 和 `推理实例` 两类入口提供模型推理能力 |
| [模型微调](./finetune/finetune_intro) | 基于 LLaMA-Factory 或 MS-Swift 框架，对基座模型进行定制化训练 |
| [模型评测](./evaluation/evaluation_create) | 基于主流评测框架，对模型进行基准测试和性能评估 |
