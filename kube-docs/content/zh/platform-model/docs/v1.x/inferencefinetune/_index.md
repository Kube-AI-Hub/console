## 功能概览

资源管理模块为大模型全生命周期提供算力支撑，涵盖以下四大功能：

| 功能 | 说明 |
|------|------|
| [Notebook 实例](./notebook/notebook_intro) | 一键创建交互式开发环境，支持 JupyterLab、VS Code、Eclipse Theia |
| [模型推理](./inference/inference_intro) | 将模型一键部署为专属推理服务，提供 Web 测试界面和标准 API |
| [模型微调](./finetune/finetune_intro) | 基于 LLaMA-Factory 或 MS-Swift 框架，对基座模型进行定制化训练 |
| [模型评测](./evaluation/evaluation_create) | 基于主流评测框架，对模型进行基准测试和性能评估 |
