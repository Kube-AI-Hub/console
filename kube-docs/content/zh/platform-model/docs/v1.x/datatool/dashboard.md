## 概述

系统仪表盘提供数据工具（DataFlow）全局运行状态的可视化视图，帮助管理员和用户快速了解各任务模块的运行情况。

## 任务统计

在**数据工具 → 系统仪表盘**页面，可查看以下三大任务模块的整体运行情况：

- **数据采集**：从外部数据源导入数据的任务
- **格式转换**：文件格式转换任务
- **数据处理**：使用算子 Pipeline 进行数据清洗/增强的任务

每个模块展示以下状态统计：

| 状态 | 说明 |
|------|------|
| 等待中 | 任务已提交，排队等待执行 |
| 执行中 | 任务正在运行 |
| 已完成 | 任务成功完成 |
| 出错 | 任务执行失败 |

## Celery 节点服务列表

仪表盘还展示所有 Celery 工作节点的运行信息：

| 字段 | 说明 |
|------|------|
| IP 地址 | 节点的网络地址 |
| 当前任务数 | 节点当前正在处理的任务数量 |
| 节点状态 | 在线 / 离线 |
| 心跳时间 | 节点最近一次心跳时间 |

管理员可在此页面监控各节点负载，并对异常节点进行管理（删除失效节点）。

## 进入数据工具

在仪表盘页面可直接点击**创建任务**按钮，快速跳转到对应模块的新建任务页面。
