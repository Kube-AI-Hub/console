## 什么是数据工具

数据工具（DataFlow）是一站式的数据处理平台，实现从数据到模型的全生命周期闭环，助力持续优化。它兼容多种数据格式和来源，支持本地文件、云端数据以及数据库的接入，并提供高效的转换与处理工具以确保数据一致性。可定制的 Pipeline 实现复杂数据的清洗和转换，结合并行处理提升效率。此外，通过集成标注系统支持多人协作，确保标注的准确性与数据质量。

数据工具与平台数据集深度集成——处理前需要先在平台创建数据集，处理结果也将直接保存至数据集仓库，实现统一的数据流转。

## 功能模块

| 模块 | 说明 |
|------|------|
| [系统仪表盘](./dashboard) | 查看数据采集、格式转换、数据处理三大任务的整体运行状态 |
| [数据采集](./data_collection) | 从 MySQL、MongoDB、文件、Hive 等来源导入数据 |
| [格式转换](./format_conversion) | 支持 Excel→JSON/CSV/Parquet、Word/PPT→Markdown 等格式转换 |
| [数据处理](./data_processing) | 使用可视化工作流编辑器，配置算子 Pipeline 进行数据清洗与增强 |
| [算法模板](./algorithm_template) | 内置和自定义数据处理算法模板，支持复用和共享 |
| [工具池](./tools) | 汇聚多种数据处理工具，支持分析、转换、生成等专项任务 |
| [数据标注](./data_labeling) | 集成 Label Studio，支持文本、图像、音频、视频等多模态数据标注 |

## 快速开始

1. 在平台数据集页面创建或选择一个数据集作为数据来源和处理结果的存储目标
2. 进入顶部导航的**数据工具**模块
3. 根据需求选择对应功能：
   - 需要导入外部数据 → [数据采集](./data_collection)
   - 需要转换文件格式 → [格式转换](./format_conversion)
   - 需要清洗/增强数据 → [数据处理](./data_processing)
   - 需要标注数据 → [数据标注](./data_labeling)
