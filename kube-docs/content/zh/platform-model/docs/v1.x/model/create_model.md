## 进入创建页面

登录平台后，点击右上角的个人头像，在下拉菜单中选择 **新建模型**，即可进入模型创建页面。也可以直接访问以下地址：

```text
https://<平台地址>/models/new
```

## 填写模型信息

在模型创建页面中，需要填写以下表单字段：

| 参数 | 是否必填 | 说明 |
|------|----------|------|
| 所有者 | 是 | 默认为当前登录用户；如果用户加入了组织，可以从下拉列表中选择该组织作为所有者 |
| 模型英文名称 | 是 | 模型仓库的唯一标识。长度 2-64 个字符，必须以字母开头，以字母或数字结尾，只能包含字母、数字、连字符（`-`）、下划线（`_`）和点号（`.`），特殊字符不能连续出现 |
| 模型别名 | 否 | 可选的显示名称，用于在页面上更友好地展示模型名称 |
| License | 是 | 选择模型使用的开源许可证类型，例如 Apache-2.0、MIT、GPL-3.0 等 |
| 模型简介 | 否 | 对模型的简短文字描述，帮助其他用户了解模型的用途 |
| 可见性 | 是 | **公开**：任何人都可以访问该模型仓库；**私有**：仅授权用户可以访问 |

填写完成后，点击 **创建模型** 按钮提交。

{{< notice note >}}
模型英文名称（即仓库名称）创建后不可修改，请谨慎填写。
{{</ notice >}}

## 创建后操作

模型仓库创建成功后，系统会自动生成一个 `README.md` 文件。建议按照[模型卡片](./model_card)的规范编辑该文件，添加模型的详细描述、使用示例、元数据标签等信息。

完成创建后，您可以继续执行以下操作：

- [上传模型文件](./upload_model)
- [编辑模型信息](./update_model)
- [下载模型](./download_models)

## 查看模型列表

平台提供以下方式浏览和查找模型：

- **模型广场**：访问 `https://<平台地址>/models` 查看所有公开模型，支持按任务类型、框架、许可证等条件筛选。
- **个人模型**：点击个人头像进入个人中心，在 **模型** 标签下查看自己创建的所有模型仓库。
- **组织模型**：进入组织页面，在 **模型** 标签下查看该组织下的所有模型仓库。
