行业大模型平台是一个开源、可信的大模型资产管理平台，帮助用户统一管理模型库、数据集、用户空间与开发环境，并提供模型推理、模型训练评测与资源管理等 LLMOps 能力。

当前平台导航的核心入口包括：

- `模型库`：统一浏览和管理模型资产。
- `模型推理`：下设 `公共推理服务` 与 `推理实例` 两个子菜单。
- `模型训练评测`：下设 `模型微调` 与 `模型评测` 两个子菜单。
- `数据集`：浏览和管理数据集资产。
- `开发环境`：进入 Notebook、VS Code、Eclipse Theia 等开发实例的创建与管理页面。
- `用户空间`：管理用户空间与部署应用。
- `资源管理`：统一查看开发环境、推理实例、微调实例和模型评测等资源。
