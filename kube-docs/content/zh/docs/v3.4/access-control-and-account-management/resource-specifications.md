## 概述

规格管理功能允许平台管理员预定义计算资源规格模板，包括 CPU、内存、磁盘和 GPU 配置。用户在创建容器实例（如 Notebook、推理实例、微调实例等）时，可直接选择预定义的规格，无需手动填写资源参数，简化操作流程并规范资源使用。

## 使用场景

- **标准化资源分配**：为不同类型的工作负载定义标准规格（如开发型、训练型、推理型）
- **简化用户操作**：用户无需了解底层资源配置细节，选择规格即可创建实例
- **成本管控**：通过预定义规格限制用户可申请的最大资源量

## 创建资源规格

1. 以平台管理员身份登录。
2. 进入**平台管理 → 规格管理**页面。
3. 点击**创建规格**按钮。
4. 填写规格信息：

| 参数 | 说明 |
|------|------|
| **规格名称** | 自定义名称，便于用户识别（如"开发-4C8G"、"训练-A100-80G"） |
| **CPU** | CPU 核数配置 |
| **内存** | 内存大小配置（GiB） |
| **GPU 型号** | 可选，指定 GPU 型号（如 A100、V100） |
| **GPU 数量** | 可选，GPU 卡数量 |
| **GPU 显存** | 可选，每张 GPU 分配的显存大小 |

5. 点击**确定**完成创建。

## 管理规格

在规格管理列表中，可以对已创建的规格进行以下操作：

- **查看**：查看规格的详细配置信息
- **编辑**：修改规格的资源配置参数
- **删除**：删除不再使用的规格

{{< notice note >}}
删除规格不会影响已使用该规格创建的实例。已运行的实例仍按原资源配置继续运行。
{{</ notice >}}

## 使用规格

规格创建完成后，用户在以下场景中可选择预定义规格：

- 创建 Notebook 开发实例时，在**资源配置**步骤选择规格
- 创建模型推理专属实例时，选择算力规格
- 创建模型微调实例时，选择 GPU 资源规格
- 创建应用空间时，选择云资源规格

选择规格后，系统会自动填充对应的 CPU、内存、GPU 等资源配置参数。
