在 Kube AI Hub 中，具有必要权限的项目用户能够执行一系列任务，例如创建各种工作负载，配置卷、密钥和 ConfigMap，设置各种发布策略，监控应用程序指标以及创建告警策略。由于 Kube AI Hub 具有极大的灵活性和兼容性，无需将任何代码植入到原生 Kubernetes 中，因此用户可以在测试、开发和生产环境快速上手 Kube AI Hub 的各种功能。
